# Template0.1.1
Configured packages such as npm, bower, sass, gulp, and more.

## Install
1. git clone https://github.com/AlexsofarBOT/Template0.1.1.git
2. npm i
3. bower i
4. gulp

## Build the project in the dist folder
* gulp build

## Installed plugins:
* gulp
* gulp-autoprefixer
* gulp-clean
* gulp-if
* gulp-minify
* gulp-responsive
* gulp-sass
* gulp-sftp
* gulp-uglify
* gulp-uncss
* gulp-useref
* wiredep
